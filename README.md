# assists-examples

本仓库是 [assistsx-js](https://github.com/ven-coder/assistsx-js) 框架的示例项目集合。

## 示例项目

### [assistsx-js-simple](./assistsx-js-simple)

assistsx-js 的入门示例项目，使用 Vite + Vue 3 集成 assistsx-js，演示在 Android 端通过 AssistsX 运行 Web 插件的完整流程，包含：

- 插件工程搭建与配置
- 基于 Step、Node 等 API 的自动化脚本编写
- 本地打包与局域网加载两种运行方式

详细说明与运行步骤见 [assistsx-js-simple/README.md](./assistsx-js-simple/README.md)。

## 关于 assistsx-js

[assistsx-js](https://github.com/ven-coder/assistsx-js) 支持在 Web 端编写、在 Android 端 [AssistsX](https://www.pgyer.com/SqGaCx8C) 中运行的自动化脚本，提供元素查找、手势执行、步骤器等能力。开发前需在手机上安装 AssistsX。

- 框架仓库与快速开始：[https://github.com/ven-coder/assistsx-js](https://github.com/ven-coder/assistsx-js)
- API 开发文档：[README-DEV.md](https://github.com/ven-coder/assistsx-js/blob/main/README-DEV.md)
