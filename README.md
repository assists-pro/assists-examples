# assists-examples

本目录（仓库根目录）用于存放与 **Assists** 相关的示例项目，便于查阅和运行各示例。

## Assists 相关示例

- **[assistsx-js-simple](./assistsx-js-simple)**：演示如何用 assistsx-js 在 AssistsX 中跑 Web 插件并编写自动化脚本。详见 [assistsx-js-simple/README.md](./assistsx-js-simple/README.md)。

- **[assistsx-log-simple](./assistsx-log-simple)**：演示 assistsx-js 的日志能力（写入、监听、上报）及浮窗与页面内两种展示方式。详见 [assistsx-log-simple/README.md](./assistsx-log-simple/README.md)。

*后续如有其他 Assists 相关示例，会在此继续列举。*
