# assists-examples

本目录（仓库根目录）用于存放与 **Assists** 相关的示例项目，便于查阅和运行各示例。

## Assists 相关示例

- **[assistsx-js-simple](./assistsx-js-simple)**：assistsx-js 框架的示例项目。使用 Vite + Vue 3 集成 [assistsx-js](https://github.com/ven-coder/assistsx-js)，演示在 Android 端通过 AssistsX 运行 Web 插件、编写自动化脚本的完整流程。详细说明与运行步骤见 [assistsx-js-simple/README.md](./assistsx-js-simple/README.md)。
